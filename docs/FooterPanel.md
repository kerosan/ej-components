### FooterPanel ###
Футер

Использование:
```
<FooterPanel/>
```
Свойства:
   
| имя          | тип                | описание                                           |
|--------------|--------------------|----------------------------------------------------|
| className    | string             | дополнительный CSS-класс                           |
| links        | ILinks             | ссылки нижнего меню неавторизованного пользователя |
| switcher     | JSX.Element        | переключатель языка                                |
CSS-классы:  

* `.ej-footer-panel`
